Hyperledger Fabric

Blockchain

  • a distribuited ledger that records all the transactions that take place on the network
  • decentralized - because it is replicated across many network participants, each of whom collaborate in its maintenance
  • append-only - using cryptographic techniques that guarantee that once a transaction has been added to the ledger it cannot be modified
  • blockchain system has shared programs to update shared ledgers.

Smart Contract

are used in a blockchain network to provide controlled access to the ledger and to enable a whole host of ledger functions (transacting, querying, etc)
not only a key mechanism for encapsulating information and keeping it simple across the network, they can also be written to allow participants to execute certain aspects of transactions automatically

Consensus

ledgers update only when transactions are approved by the appropriate participants, and that when ledgers do update, they update with the same transactions in the same order
